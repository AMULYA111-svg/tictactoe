# Tic Tac Toe Game using Python

This is a GUI-based Tic Tac Toe game built using Python and Tkinter.

## Features
- Dark-themed attractive UI
- Player vs Player mode
- Winner detection
- Draw detection
- Score tracking
- Replay option after each game

## Tech Used
- Python
- Tkinter

## How to Run
```bash
py tictactoe.py
